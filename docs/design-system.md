# UI 视觉系统

参考图同时作为品牌头像资产和风格输入。项目内使用 `public/brand-avatar.png` 作为本地 logo/avatar；其余 UI 视觉继续从参考图提炼，不额外复制人物图到其他装饰位置。

## 提炼结果

- 主背景：高饱和橙黄到粉橙渐变。
- 主面板：接近黑紫的半透明玻璃面板。
- 强调色：蓝色镜片感 `#00D7FF`、紫色高光 `#7B2CFF`、粉色高光 `#FF3D8B`。
- 图形语言：圆角矩形、霓虹描边、强对比、局部发光。
- 使用方式：适合启动页、侧边栏品牌区、状态图标和按钮 hover；密集表格保持高对比和可读性。

## 禁止项

- 不把头像重复铺满背景或作为大面积装饰。
- 不把数据表做成大面积霓虹，避免影响可读性。
- 不在核心任务流里加入装饰性欢迎页。
