# 按需扫描设计

## 默认模式

默认是快速转存模式：

```text
转存 -> 列表 -> 分类 -> 重命名 -> 移动 -> 创建分享
```

快速模式下：

- 不下载文件样本
- 不初始化 OCR
- 不检查 OCR 模型
- 不检查 ffmpeg
- 不视频抽帧
- 不生成清理副本

## 模式

| 模式 | 用途 | 默认扫描 |
| --- | --- | --- |
| off | 最快完成转存和分享 | 全部关闭 |
| standard | 轻量发现明显风险 | QR/OCR/联系方式/引流词 |
| deep | 发布前深度检查 | QR/OCR/联系方式/引流词/水印/视频抽帧 |

## ShareTiming

- `share_immediately`：默认，先完成分享，扫描后台进行。
- `share_after_scan_passed`：扫描完成且无高风险后再创建分享；有高风险则暂停确认。

## 清理副本

只有勾选“生成清理副本”时才执行清理。输出副本，不覆盖 raw 文件，不删除网盘文件。
